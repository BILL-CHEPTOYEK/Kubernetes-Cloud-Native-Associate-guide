---

# Certification Overview

The **KCNA certification** validates foundational knowledge of Kubernetes and cloud native technologies.

**Certification Provider:** CNCF  
**Course:** LFS250 – Kubernetes and Cloud Native Essentials  
**Exam Format:** Multiple choice  

**Exam Details:**

* Questions: 60
* Duration: 90 minutes
* Passing Score: ~75%
* Attempts: 1 attempt (additional attempts require a new voucher)

**Core Domains Covered:**

1. Kubernetes Fundamentals
2. Container Orchestration
3. Cloud Native Architecture
4. Cloud Native Observability
5. Cloud Native Application Delivery

# Hands-On Tools

Learning Kubernetes effectively requires hands-on practice.

* **Minikube** – local Kubernetes clusters

# Study Methodology

The repository follows an active learning cycle:

**Learn → Practice → Summarize → Test**

For each topic:

1. Study the concept
2. Perform hands-on labs
3. Summarize the material
4. Solve practice questions

This method improves understanding and exam readiness.
