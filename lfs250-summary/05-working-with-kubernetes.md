# Working with Kubernetes

## Introduction

### Learning Objectives

- Understand Kubernetes objects and how to interact with the cluster
- Use `kubectl` to inspect and manipulate resources
- Understand pods, workloads, services, volumes, config objects, and autoscaling

## Topics Covered

- Kubernetes objects and `kubectl` (demo)
- Pod concept and lifecycle
- Workload objects and demos
- Networking objects and ingress vs Gateway API
- NetworkPolicy
- Volumes and storage
- Configuration objects and autoscaling
- Scheduling and security
